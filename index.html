Olá,

este é o site oficial da ferramenta de automação de testes funcionais Demoiselle Behave, a ideia é que através deste site seja possível conhecer a ferramenta, dar início a um projeto e tirar dúvidas na documentação.

Links Úteis

  • Documentação: Aprenda agora!
  • Fórum: Abra um issue para discutirmos um assunto
  • Bugs e Sugestões: Abra um issue para submeter e acompanhar Bugs, enviar dúvidas, propor melhorias ou novas funcionalidades
  • Catálogo de Arquétipos Releases: Para versões estáveis
  • Catálogo de Arquétipos Snapshot: Para versões de desenvolvimento (snapshot)
